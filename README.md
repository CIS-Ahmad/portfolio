# 💼 Personal Portfolio

A modern and responsive personal portfolio website built using **HTML5** and **CSS3**. This project showcases my skills, projects, and contact information through a clean, professional interface.

## 🌐 Live Demo

👉 **Website:** https://cis-ahmad.github.io/portfolio/

---

## 📖 About

This portfolio was created to serve as my personal website where I can introduce myself, present my technical skills, and showcase the projects I have developed throughout my learning journey.

The website focuses on simplicity, responsiveness, and a clean user experience while following modern web design principles.

---

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Fixed sidebar navigation on desktop
* Mobile-friendly top navigation
* Animated hero section with CSS typing effect
* About Me section
* Skills section
* Projects showcase
* Contact section
* Download CV button
* Social media and GitHub links
* Smooth scrolling navigation
* Modern dark theme UI

---
